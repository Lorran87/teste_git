1. `git init` - Inicia um novo repositório Git em um diretório.

2. `git add` - Adiciona alterações de arquivos ao índice de preparação.

3. `git commit` - Cria um novo registro de versão (commit) com as alterações preparadas.

4. `git status` - Mostra o estado atual dos arquivos no repositório.

5. `git log` - Exibe o histórico de commits.

6. `git branch` - Lista, cria ou deleta branches.

7. `git checkout` - Muda entre branches ou restaure arquivos.

8. `git merge` - Combina as mudanças de um branch em outro.

9. `git remote add` - Adiciona um repositório remoto.

10. `git push` - Envia commits para um repositório remoto.

11. `git clone` - Cria uma cópia de um repositório remoto.
 
`git config` para configurar informações do usuário,

`git rm` para remover arquivos do repositório, e

`git remote -v` para listar repositórios remotos configurados.
